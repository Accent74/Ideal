Построение отчетов
Чтобы построить отчет, шелкните по его наименованию. Отчеты будут строиться за _period_title

Внимание! В текущем периоде имеется 10 нерасчитанных документов!
Данные в отчетах могут быть неполными!
Отчеты
Сводная ведомость по заработной плате
Сводная ведомость по заработной плате показывает суммарные начисления, удержания по всему предприятию.
Проводки в "Главную книгу"
Сводная ведомость показывает проводки по начислениям и удержаниям, сгруппированные по бухгалтерским счетам для занесения в "Главную книгу".
Отчисления в фонды
Ведомость показывает суммы по отчислениям с фонда оплаты труда, и базовые суммы для расчета этих отчислений.
Остатки за период
Ведомость показывает всех сотрудников, для которых имеется остаток на конец периода (либо долг предприятия, либо долг сотрудника).
