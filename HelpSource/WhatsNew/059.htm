Версия 7.0.059 (12 марта 2011)

Содержание <<Предыдущая Следующая>>  

Ошибки
  • Исправлена ошибка в методе расчете НДФЛ для случаев превышения суммы зарплаты предела.
  • Исправлена ошибка в методе расчете льготы по НДФЛ.
  • Исправлена ошибка в методе начисления, из-за которой невозможно было ввести информацию об отработанном времени в часах.
Функциональность
  • Добавлены методы для расчета удержаний для шахтеров.
  • Добавлена форма первичного документа Обязательные удержания с 010111 (шахтеры)  для расчета удержания для шахтеров.
  • Добавлена форма первичного документа Аванс развернуто с 01012011 (шахтеры) для расчета аванса и удержаний по нему для шахтеров.
  • В форму первичного документа Обязательные удержания с 010111 внесены исправления в соответствиями с изменениями метода расчета льготы по НДФЛ.
  • Добавлен отчет Податкова декларация з податку на доходи фiзичных осiб.
