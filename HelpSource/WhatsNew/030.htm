Версия 7.0.030 (14 ноября 2006)

Содержание <<Предыдущая Следующая>>  
Ошибки 
  • Исправлен диалог, который записывал данные из 1ДФ в DBF файл с учетом изменений в форме отчета.
  • В форме Компенсация неиспользованного отпуска исправлена ошибка деления на ноль, если зарплата за расчетный период не начислялась.
  • В форме Расчет отпускных развернуто исправлена ошибка, из-за которой сумма в колонке "Начисленная зарплата" округлялась до целого числа. 
  • В форме Больничные развернуто исправлена ошибка в тексте "Итого часов к расчету".
  • Исправлена ошибка в форме Расчет отпускных развернуто из-за которой при сохранении документа не сохранялась рассчитанная сумма средней заработной платы.
Функционал 
  • Добавлена возможность автоматического расчета суммы индексации заработной платы.
  • Добавлена новая таблица, где размещается информация о коэффициентах индексации.
  • В свойства сотрудника добавлено новое свойство "Базовый месяц для индексации".
  • Добавлен метод 091а для расчета индексации.
  • Добавлена форма первичного документа Индексация (автоматически), в которой производится расчет индексации .
  • В форме 1ДФ добавлена возможность указывать признак (колонка 9) и тип отчета ("Отчетный", "Новый отчетный" ...)
  • В форме Компенсация неиспользованного отпуска удалено поле "По" за ненадобностью.
  • В форме Больничные развернуто добавлено поле для подписи Бухгалтера.
  • В метод 3010 внесены изменения, которые учитывают период, за который выплачивается зарплата.
  • Обновлена 1ДФ. Теперь данные о льготах попадают разными строками
  • В форму Отпускные развернуто добавлена возможность исключать из расчета средней заработной платы месяцы по усмотрению бухгалтера. Например, месяцы, в которых работник брал отпуск за свой счет.
  • В форме Больничные развернуто добавлена возможность использовать управлять используемыми методами. Для этого используется параметр папки "Код(ы) начислений/удержаний" в котором используемые в форме методы можно перечислить через запятую. Первый метод используется для начислений за счет предприятия, второй за счет соцстраха. Если какой либо метод не указан или не найден, то соответствующее поле для ввода дней блокируется.
  • Добавлен отчет Ведомость начислений, который строится за месяц и отображает все начисления и удержания для всех сотрудников с итогами по подразделениям, а также начисления на ФОТ. Отчет размещен в списке отчетов "корня" дерева корреспондентов.
