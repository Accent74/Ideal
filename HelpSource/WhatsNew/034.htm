Версия 7.0.034 (23 февраля 2007)

Содержание <<Предыдущая Следующая>>  
Ошибки
  • Исправлен метод 2100п Подоходный налог.
  • Исправлен метод 2220п Алименты для ввода вручную суммы.
  • исправлен метод 0530m Компенсация отпускных.
  • Изменен метод 3010п Выплата зарплаты таким образом, чтобы он учитывал уже выданную в текущем периоде заработную плату.
  • В формах первичных документов Табель (оклад), Табель (час) и Табель (вручную) исправлена ошибка, из-за которой при удалении сотрудника из списка могла пропадать полоса прокрутки.
Функционал 
  • В формах первичных документов для ввода информации внесены изменения, которые позволяют устанавливать период, за который проводится расчет, сразу для всех проводок.
  • В формы первичных документов Табель (оклад), Табель (час), Табель (вручную) и Идеал сдельная добавлена возможность сортировки списка по имени и табельному номеру (если предусмотрено его отображение).
  • В формы первичных документов Табель (час) и Табель (вручную) добавлено отображение итогов по столбцам формы .
  • В форме Персонификация на первой странице добавлена возможность сортировать сотрудников по алфавиту.
  • Отчет в Расчет сумм страховых взносов по обязательному государственному пенсионному страхованию приведен в соответствие с изменениями в законодательстве.
