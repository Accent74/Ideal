Функциональность
  • Добавлен экспорт проводок в программу Акцент. Встроенный экспорт после изменений в законодательстве перестал быть актуальным. Для осуществления экспорта необходимо создать отчет Импорт проводок в Акцент, который строится в разделе Отчеты по предприятию Моя фирма.
  • Изменился метод 0116m Сдельная оплата (вручную). Теперь счет по дебету начисленной суммы проставляется автоматически
  • Добавлены ставки коэффициента индексации для расчета индексации на 3 квартал
