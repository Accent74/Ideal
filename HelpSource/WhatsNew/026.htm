Версия 7.0.026 (21 июня 2006)

Содержание <<Предыдущая Следующая>>  

Ошибки
  • Исправлена ошибка в процедуре загрузки сотрудников подразделения в формы первичных документов. Если в списке сотрудников были почасовики и окладники, то ранее появлялись пустые строки.
  • Исправлена ошибка в обновлениях с 22 по 25 версию, которая проявлялась только на SQL версии программы.
Функциональность 
  • Приведен в соответствие с изменениями в законодательстве алгоритм расчета среднедневной заработной платы для расчета больничного. Для работы формы добавлены коды начислений : 0024 "Разовые выплаты", 0025 "Больничные", 0026 "Отпускные" и 0027 "Начисления". В зависимостях этих кодов требуется указать коды, данные по которым должны попадать в одноименные столбцы формы для расчета.
  • В ведомости 1ДФ теперь правильно отражается выданная заработная плата. Для этого добавлены коды : 0028 "1ДФ начисления" и 0029 "1ДФ удержания". В зависимостях этих кодов нужно указать те коды, данные по которым будут расчитываться в колонках 3а и 4а формы 1ДФ.
  • Откорректирован внешний вид отчёта Розрахунок збору пенсiйного страхування (Додаток 23).
