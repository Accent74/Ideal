Версия 7.0.424 (08 января 2004)

Содержание <<Предыдущая Следующая>>  
Акцент 7.0
  • Изменен механизм запуска программы под управлением OLE-automation. Теперь каждый вызов CreateObject с аргументом Accent7.Workarea (или Accent7s.Workarea для SQL-версии) открывает новый экземпляр программы. Кроме того, теперь можно программно запускать новые копии Акцент из форм, диалогов, таблиц и модулей. Такой подход позволит упростить разработку программ импорта-экспорта.
    Обратите внимание, что открывать БД под управлением OLE-automation следует с использованием метода OpenDatabase2.
VBScript
  • В объект Workarea добавлен метод OpenDatabase2, который предназначен для открытия базы данных при работе программы под управлением OLE-automation.
Акцент-зарплата 7.0
  • Выпущена release-версия программы Акцент-зарплата 7.0 Для использования программы нужно получить файл лицензии.
  • Появился мастер полного расчета заработной платы. Запускается из основного меню. Позволяет выполнять массовые операции по созданию и расчету проводок.
  • Теперь файл stdpayroll.chm (содержимое этого файла показывается в окне "Задачи") сначала ищется с использованием ard-файла. Если файл не найден, то берется файл, который находится в папке с программой.
Акцент-администратор 7.0
  • Выпущена release-версия программы Акцент-администратор 7.0 Для использования программы теперь нужно получить файл лицензии. Обновления будут доступны на www-сайте.
