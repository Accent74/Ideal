Версия 7.0.013 (6 июля 2005)

Содержание <<Предыдущая Следующая>>  

Общая информация

  • Добавлена форма подробных отпускных, в ней наглядно расчитывается среднедневной тариф за 12 отработанных месяцев
  • В документе Табель (Оклад) у сотрудников из разных подразделений теперь корректно записывается в проводку их родное подразделение
  • В окно статуса выводятся ошибки при заполнении табеля
  • Ведомость обязательных удержаний: при ручном заполнении у сотрудников из разных подразделений теперь после расчёта в проводке указывается их родное подразделение
  • Изменена форма 1ДФ. Теперь значения процентов для ФОТ (27 признак дохода) берутся непосредственно из кодов ФОТ

Формы

  • Добавлена форма начисления Отпускные развёрнуто, её следует переместить в Начисления - За отсутствие

Инструкция

  • Для автоматического расчёта среднедневного тарифа в форме Отпускные равёрнуто необходимо в свойствах кода 0110 Основная зарплата поставить галочку напротив Использовать для расчёта среднего (дни и часы)
  • В зависимость кода 0510 Отпускные добавить код 0110 Основная зарплата и другие коды по надобности для расчёта среднедневного тарифа для Отпускных
