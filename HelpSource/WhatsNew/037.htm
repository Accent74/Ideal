Версия 7.0.037 (23 октября 2007)

Содержание <<Предыдущая Следующая>>  
Ошибки
  • В форме первичного документа Форма 1ДФ исправлена ошибка из-за которой при "Импорте физ. лиц" строки в таблицу добавлялись только на втором листе.
  • Исправлена ошибка в Общих подпрограммах из-за которой для сотрудника принятого или уволенного в месяце расчета при определенных условиях могло возвращаться неправильное количество максимально отработанных дней.
  • Исправлена ошибка в методе 2220п Алименты из-за которой могли сбрасываться данные по % алиментов. Последствия ошибки носили только визуальный характер.
  • Исправлена ошибка в методе 0801-01 Льгота по подоходному налогу из-за которой в расчет не бралась ранее рассчитанная сумма льготы.
  • Исправлена ошибка в форме первичного документа За совмещение исправлена ошибка из-за которой кнопки Сохранить и Рассчитать оставались активными не зависимо от состояния документа.
  • Исправлена ошибка в форме первичного документа Ведомость начисления заработной платы из-за которой в расчет мог не приниматься сотрудник, принятый и уволенный в месяце расчета.
Функционал 
  • Изменен бланк отчета Расчетная ведомость по несчастному случаю на производстве (файл ФСС НСП.ash).
  • В форме первичного документа Табель (час) добавлены итоги по часам График.
  • В формы первичных документов Табель (час), Табель (вручную), Больничные по беременности (авт), Больничные по беременности(вручную), Больничные по травме (авт), Больничные по уходу за ребёнком (авт), Больничные по травме (вручную), Больничные по уходу за ребенком(вручную), Больничные (вручную), Больничные развернуто после 01.06.2006 (авт), Компенсация за неиспользованный отпуск, Отпускные (вручную), Отпускные развёрнуто, Обязательные с 01.07, Прочие удержания добавлен табельный номер сотрудника для ввода и отображения.
  • Исправлены методы : Больничные за счет СС, Больничные по беременности, Больничные по уходу за ребенком, Больничные по травме. Теперь счет по Дт устанавливается из свойств метода.
  • В методе 0420п счет по Дт используется из свойств кода.
  • В методе 091 счет по Дт используется из свойств подразделения
  • В форме первичного документа 1ДФ добавлена возможность выбирать XML файл для импорта.
  • Форма первичного документа Идеал_Отпускные_Развёрнуто.afm рассчитывает отпускные по зависимостям метода 0510.
