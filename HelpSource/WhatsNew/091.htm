Версия 7.0.91 (12 марта 2015 г.)

Содержание <<Предыдущая Следующая>>  

Функционал
  • Добавлен отчет Отчет по ФОТ. Отчет показывает начисления на ФОТ сотрудника и начисление на ФОТ предприятия по каждому сотруднику.
  • Для правильного расчета начислений на ФОТ добавлены методы у кодов, ответственных за ФОТ по предприятию : 2000-01 ЕСВ базовый, 2001-01 ЕСВ, инвалиды (8,41%, ФОТ), 2002-01 ЕСВ, больничный (33,2% для всех, ФОТ), 2003-01 ЕСВ ГПД (34,7%, ФОТ), 2004-01 ЕСВ декретные (33.2%, ФОТ). Добавлены необходимые зависимости.
  • Добавлены обязательные начисления по сотруднику 2450а ЕСВ (декретные, аванс), 2460 ЕСВ (инвалиды), 2460а ЕСВ (инвалиды, аванс).
  • Добавлена форма для начисления ФОТ по предприятию. См. папку ФОТ в папке Начисления на ФОТ. Начисления на ФОТ следует проводить после ввода всех начислений.
  • Форма начисления обязательных удержаний изменена в связи с увеличением количества удержаний. На отдельную страницу перенесены удержания по ЕСВ для инвалидов и ЕСВ для декретных.
  • Форма начисления аванса изменена в связи с увеличением количества удержаний. Документ разбит на две страницы. На первой начисления, на второй удержания по авансу.
