Версия 7.0.036 (27 июня 2007)

Содержание <<Предыдущая Следующая>>  
Ошибки
  • В платежной ведомости исправлена ошибка, которая возникала, если попытаться изменить период За , не указав подразделение. Теперь выбор периода заблокирован до того момента, пока не будет указано подразделение.
  • Исправлена ошибка в форме первичного документа Справка о доходах из-за которой в сохраненном ранее документе невозможно было изменить и запомнить год, за который выдавалась справка.
  • Исправлена неточность в форме первичного документа Отпускные развернуто и Компенсация за неиспользованный отпуск из-за которой в итоговой строке по колонке "кол-во календарных дней" отображалось неправильное значение.
  • Отчет ФСС ВПТ :
    • Суммы "Материальне забезпечення" считаются нарастающим итогом по всем кварталам с первого по текущий
    • В графу (80, 17) и (80, 21) теперь вносятся данные по методу 0420 "Больничные соцстрах" Раньше вносилось 0420+0450
  • Отчет ФСС НСП 
    • Исправлено на все месяцы по кварталу а так же отображение данных по всем предыдущим кварталам
    • На втором листе не рассчитывались больничные по методу 0450 Добавлены нарастающим итогом с расчетом кол-ва потерпевших, дней, суммы
    • Добавлен расчет ФОТ на листе 2, табл.4. ФОТ (по квартально) берется аналогично листу 1, табл.1 только без учета максимальной границы.
  • Исправлена ошибка в форме первичного документа Расчет больничных (развернуто) из-за которой при смене работника данные в таблице не пересчитывались.
  • В формах первичных документов Табель (оклад), Табель (час) и Табель (вручную) исправлены положения некоторых элементов на листе для многостраничной печати.
  • Исправлена ошибка в форме первичного документа Аванс развернуто из-за которой при ручном вводе сотрудника, не всегда правильно формировалась ведомость выплаты.
  • Отчет ФСС БР
    • отчет теперь строится по реестру проводок и учитывает тарифы если было два начисления в периоде Когда
    • Аналогично изменен расчет для инвалидов (только по каждому сотруднику) если сотрудник инвалид, то по нему строится реестр проводок и выбираются проводки по коду 2130.
    • Не учитываются авансы при расчете этих значений
  • Исправлен отчет Розрахунок збору пенсiйного страхування (Додаток 23).
    • В графу 1.1.2. теперь вносятся данные по методам (сумма): 0420 "Больничные соц.страх", 0440 "Больничные по уходу за ребенком (СС)", 0450 "Больничные по травме".
    • теперь учитываются удержания по авансам.
    • Добавлена сумма удержаний по коду "2110а Пенсионный фонд (аванс)".
Функционал 
  • В методы начисления налогов добавлено округление сумм там, где его ранее не было.
  • В отчёте Ведомость начислений :
    • теперь не отображаются коды типа выплаты.
    • выбираются только проводки, у которых месяц За когда совпадает с выбранным для отчёта месяцем.
  • У метода  0801(ЛЬГОТА) из зависимостей убран метод 0520 (отпускные будущего периода), поскольку льгота к отпускным ошибочно применялась дважды.
  • Обновлены методы 0420п, 0450w
  • Добавлена форма первичного документа Идеал_Больничные_Развернуто2ФНС для регистрации больничных по травме. Форма подключена к папке Больничные по травме.
  • Добавлена форма первичного документа Идеал_Больничные_уход за ребенком Развернуто2 для регистрации больничного по уходу за ребенком за счет соц.страха. Он подключен к папке Больничные по уходу за ребенком (СС).
  • Добавлена новая форма первичного документа Идеал_Больничные_Пособие по беременности Развернуто2.  Форма подключена к папке Больничные по беременности.
  • В отчёте Ведомость начислений коды Долг предприятия и Долг сотрудника теперь не отображаются.
  • Внесены исправления в алгоритм расчета рабочих дней для сотрудников, принятых и уволенных в одном месяце (общие подпрограммы).
  • В форму документа Компенсация за неиспользованный отпуск добавлена возможность указывать месяцы, которые должны попадать в расчет.
