Версия 7.0.012 (30 мая 2005)

Содержание <<Предыдущая Следующая>>  

Общая информация

  • Добавлена форма подробных больничных, в ней наглядно расчитывается среднедневной тариф за 6 отработанных месяцев
  • В форму Алименты попадают теперь только работающие сотрудники
  • В платёжной ведомости можно указать: выдавать с копейками или без
  • Ведомость обязательных удержаний: при заполнении в неё попадают только те сотрудники, у которых есть начисления в текущем периоде
  • Добавлено два отчёта в фонды БР м ВПТ в виде Sheet. Подключите их в ручную.

Формы

  • Добавлена форма начисления Больничные развёрнуто, её следует переместить в Начисления - За отсутствие

Отчёты

  • Добавлен отчёт ФСС БР (sheet)
  • Добавлен отчёт ФСС ВПТ (sheet)

Инструкция

  • Для автоматического расчёта среднедневного тарифа в форме Больничные равёрнуто необходимо в свойствах кода 0110 Основная зарплата поставить галочку напротив Использовать для расчёта среднего (дни и часы)
  • В зависимость кода 0410 Больничные за счёт предприятия добавить код 0110 Основная зарплата и другие коды по надобности для расчёта среднедневного тарифа для Больничных
