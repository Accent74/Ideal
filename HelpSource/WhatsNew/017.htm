Версия 7.0.017 (31 октября 2005)

Содержание <<Предыдущая Следующая>>  

Общая информация

  • Добавлен Отчёт по методу. Показывает сумму начислений по выбранному методу и базу налогооблажения (зависимости) для кода этого метода.
  • Исправлена ошибка в Табель (оклад). Нельзя было вручную корректировать процент премии.
  • Все документы создаются по умолчанию последним днём месяца
  • В форме Сдельная оплата труда теперь есть дата создания документа
  • Добавлен Отчёт реестр кодов программы Акцент-Зарплата

Формы

  • Форма Табель (оклад). После автоматического заполнения теперь можно изменять процент премии, который заполняется автоматически из свойств сотрудника
  • Форма Сдельная оплата труда. Теперь можно указать дату создания документа. Раньше было нельзя из за отсутствия соответствующего элемента на форме

Отчёты

  • Добавлен отчёт: Реестр кодов. Отчёт строит реестр по всем кодам начислений/удержаний которые используются в программе и отчётах
  • Добавлен отчёт: Отчёт по методу. Отчёт показывает базу налогооблажения (зависимость) по выбранному методу в разрезе корреспондентов. Полезен для отображения сумм по отдельным методам, например Пенсионный 2% и 1%

Методы

  • Изменён метод начисления Премии. Теперь если значение меняли в Табеле, то процент не восстанавливается из свойст сотрудника, а в расчёт берётся тот что ввели в форму

Инструкция

  • Отчёт Реестр кодов - перед работой надо подключить к необходимому подразделению.
  • Отчёт по методу - перед работой надо подключить к необходимому подразделению.
