1) создать проект
2) скомпилировать проект в chm-файл
3) поместить chm-файл в папку, прописанную в "Путях расположения файлов"
4) в БД в таблице JOBS прописать где требуется свою страницу из созданного chm-файла
запрос:

UPDATE JOBS
SET JOB_HLINK = 'dzgpr1.chm::/job_documents.htm'
WHERE JOB_NAME = 'Работа с документами'

Примечание: в запросе нужно указать "ваши" имя chm-файла и имя htm-страницы
